# «if..else»-ը փոխարինեք «?»-ով:

Վերաշարադրեք `if..else`-ը՝ օգտագործելով մի քանի «եռակի» օպերատորներ `?`:

Ընթեռնելիության համար խորհուրդ է տրվում կոդը բաժանել մի քանի տողերի:

```js
let message;

if (login == 'Աշխատակից') {
  message = 'Ողջույն';
} else if (login == 'Տնօրեն') {
  message = 'Ողջույններ';
} else if (login == '') {
  message = 'Մուտք չկա';
} else {
  message = '';
}
```
